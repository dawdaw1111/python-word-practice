# PyWord Lite - Python 常用单词拼写练习

`PyWord Lite` 是一个离线可用的 Python 英文单词练习工具。  
学习者通过“看中文提示 -> 拼英文单词”的方式，边学编程边记住高频术语。

## 内容规模

- 总词量：50
- 分类：
- `基础入门`
- `数据类型`
- `流程控制`
- `函数基础`
- `常见报错词`

## 功能亮点

- 首页学习概览（已学、已掌握、错题等）
- 分类闯关与继续学习
- 提交判题与即时反馈
- 错题本复习（支持清理已掌握）
- 统计页（完成率、正确率、分类进度）
- 自动保存进度（`localStorage`）

## 快速开始

1. 进入项目目录
2. 直接用浏览器打开 `index.html`

或启动本地静态服务：

```bash
python -m http.server 8000
```

访问：`http://127.0.0.1:8000`

## 项目结构

- `index.html`：页面结构
- `style.css`：界面样式
- `words.js`：单词库数据
- `app.js`：学习流程与状态管理
- `mvp.md`：产品设计与 MVP 说明

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript（无框架）
